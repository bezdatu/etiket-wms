# Reference Samples

Локальная папка для эталонных изображений конфликтных этикеток.

## Рекомендуемая структура
- `reference-samples/<ambiguity-group>/<product-name>/front-01.jpg`
- `reference-samples/<ambiguity-group>/<product-name>/front-02.jpg`
- `reference-samples/<ambiguity-group>/<product-name>/angle-01.jpg`
- `reference-samples/<ambiguity-group>/<product-name>/blur-01.jpg`

## Правила
1. Один ambiguity group на набор очень похожих этикеток.
2. Для каждого товара минимум 3-5 кадров.
3. Добавлять хорошие и плохие условия: свет, угол, небольшое размытие.
4. Для benchmark файлов использовать формат имени:
   `PRODUCT_NAME__case-01.jpg`
