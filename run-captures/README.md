# Run Captures

Эта папка нужна для каждого ручного прогона.

Правило:

1. После изменения кода версия на сайте поднимается: `V1`, `V2`, `V3` и так далее.
2. Для каждого прогона сохраняется скриншот экрана Safari.
   Сохраняем именно окно приложения/сайта, а не весь рабочий стол.
3. Рядом сохраняется короткий текстовый файл с анализом:
   - какая версия была на экране
   - что сработало
   - что не сработало
   - что менять следующим шагом

Именование:

- `vN-run-01.png`
- `vN-run-01.md`

Сейчас папка используется только для визуальной проверки рамки и круга.
